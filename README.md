# Usage
<!-- usage -->
```sh-session
$ npm install -g test-cli
$ test-cli COMMAND
running command...
$ test-cli (--version)
test-cli/0.4.27 darwin-arm64 node-v20.11.1
$ test-cli --help [COMMAND]
USAGE
  $ test-cli COMMAND
...
```

## `test-cli`

Update installed plugins.

```
USAGE
  $ test-cli  [-h] [-v]

FLAGS
  -h, --help     Show CLI help.
  -v, --verbose

DESCRIPTION
  Update installed plugins.
```
